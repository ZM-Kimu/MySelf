# 老技师网络攻占器

一个用于 Windows 平台的图形化工具，用于单个IP下网络速率受限的区域。为自己的主机增加局域网 IP、开启本地代理，并将其设置为系统代理，方便进行多线程高速下载等操作。**不保证可用性。**

## 功能特性

- 扫描当前子网，获取空闲 IP 地址。
- 批量设置多个 IP 到本地网络适配器。
- 自动调用 `dispatch.exe` 作为代理服务端，并支持 `socks` 与 `http` 协议。
- 自动将代理设置为系统代理（可选）。
- 提供系统托盘图标，支持隐藏主窗口与退出。

## 依赖环境

- Python 3.x
- Windows 系统
- 以下 Python 第三方库：
  - `ping3`
  - `psutil`
  - `pystray`
  - `Pillow`

## 文件说明

| 文件名               | 说明                                    |
| -------------------- | --------------------------------------- |
| `LaojishiNetwork.py` | 主程序，包含 Tkinter 图形界面与功能逻辑 |
| `networkDelivery.py` | 网络功能模块，如设置 IP、代理、扫描等   |
| `pingHost.py`        | 基于 `ping3` 的 IP 可达性检测函数       |
| `icon.ico/png`       | 程序图标，用于图形界面与托盘图标显示    |
| `dispatch.exe`       | 外部代理服务器程序，基于Node.js         |

## 使用方法

1. 安装依赖：`pip install -r requirements.txt`
2. 启动程序：`python LaojishiNetwork.py`
3. 在界面中设置端口、协议、攻占数量，并选择是否自动设置为系统代理。
4. 点击【启动！】，程序会扫描空闲 IP，添加到本地适配器并自动运行代理服务。

## 注意事项

- 必须以管理员权限运行以便执行 IP 和注册表修改操作。
- 程序依赖于 `dispatch.exe` 可执行文件，请确保其放在当前目录。
- 推荐使用 `pyinstaller` 进行打包，例如：

```bash
pyinstaller LaojishiNetwork.spec
```
