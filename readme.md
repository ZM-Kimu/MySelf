# What is this?

这是一个用以存放个人工作的仓库，这些是学习中所制作的项目，并非总是可用。

## 项目目录

### 1. [AtomGenerator](./AtomGenerator) —— 基于 Blender 的原子模型生成器
使用 Blender 创建 3D 原子模型的工具，能够生成不同元素且包含动画的可视化模型。

### 2. [backend-dcoa](./backend-dcoa) —— 后端项目（DCOA）
一个后端系统，包含 MVC 结构的 `controllers`、`models`、`views`，以及 `utils`、`modules` 等核心功能模块。提供 API 服务，支持数据库迁移与用户管理。

### 3. [backend-IE](./backend-IE) —— 后端项目（IE）
另一个后端项目，包含 `controllers`、`models`、`routes` 及 `resources` 目录，提供 API 及图片资源管理功能。

### 4. [JDProductSpider](./JDProductSpider) —— 京东商品爬虫
一个用于爬取京东商品信息的 Python 爬虫脚本，可获取商品名称、价格、评论等数据，并支持存储到本地或数据库。

### 5. [NLPDatabase](./NLPDatabase) —— 自然语言处理数据库
通过自然语言操作数据库，仅供娱乐。

### 6. [OpenaiApiTUI](./OpenaiApiTUI) —— OpenAI 风格 API 访问工具（TUI）
一个终端用户界面（TUI）工具，用于访问和测试 OpenAI 风格的 API，用于 GPT 相关应用的开发与测试。

### 7. [PythonLearning](./PythonLearning) —— Python 学习笔记与代码
用于存放学习 Python 过程中编写的代码。

### 8. [WallpaperEngineSpineViewer](./WallpaperEngineSpineViewer) —— Wallpaper Engine 网页 Spine 视图工具
一个用于查看 Spine 动画的网页工具，包含 `assets`、`css`、`js` 及 `lib` 目录，支持光标跟随等特效。

### 9. [MultiIPBandrateAcceleratorProxy](./MultiIPBandrateAcceleratorProxy) —— 将网络适配器绑定多个 IP 以加速网络
一个用于 Windows 平台的图形化工具，用于单个 IP 下网络速率受限的区域。为自己的主机增加局域网 IP 地址以加速。
